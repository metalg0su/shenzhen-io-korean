* *x*와 *y*는 조이스틱에 연결된 단순 입력입니다.
* *a*와 *b*는 각각 버튼에 연결된 단순 입력입니다.
* *무선-rx*는 무선 수신기에 연결된 논블록킹 XBus 입력입니다.
* *무선-tx*는 무선 송신기에 연결된 논블록킹 XBus 출력입니다.
* 데이터 패킷이 무선 장비를 통해 게임 콘솔로 들어오면, 가장 최근에 입력한 값으로 다음과 같이 3 개의 데이터 패킷 형태로 응답하세요.

#############

|  |  | *a*가 0이고 *b*가 0이면, 0
| *x* | *y* | *a*가 100이고 *b*가 0이면, 1
|  |  | *a*가 0이고 *b*가 100이면, 2
|  |  | *a*가 100이고 *b*가 100이면, 3
